# context/

Operational session state. Files here are short-lived working memory.

| File | Purpose |
|------|---------|
| `current-task.md` | Active task context — what we're working on right now |
| `session-checkpoint.md` | End-of-session narrative — where we left off |
| `blockers.md` | Active blockers and open dependencies |
| `pulse-config.md` | Configuration for scheduled research pulses |
| `coordination/` | Agent coordination files (created on demand) |
| `archive/` | Expired context snapshots |
