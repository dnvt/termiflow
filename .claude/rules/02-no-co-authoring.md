# No Co-Authoring or Signatures

NEVER add to commits, commit messages, PRs, or PR messages:

- `Co-Authored-By:` lines
- `Signed-off-by:` lines
- Signature blocks or attribution
- "Generated with", "Created by", "Assisted by" messages
- AI attribution of any kind
- Emoji signatures or watermarks

The code speaks for itself. The commits are the developer's work.
