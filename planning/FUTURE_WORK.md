# Areas for Improvement / Future Work

* **TUI Implementation**: The main feature of the project, the interactive TUI, is not yet implemented. This will be the next major step to realize the project's full vision.
* **Expanded Mermaid Support**: The current Mermaid support is limited. Adding support for features like subgraphs, edge labels, and more node shapes would significantly increase the tool's utility.
* **Junction Rendering**: The specification notes that junction rendering is incomplete. Finishing this would improve the appearance of more complex graphs.
* **Debugging Tools**: Implementing the --debug-layout flag, as mentioned in the spec, would be beneficial for future development and debugging.
